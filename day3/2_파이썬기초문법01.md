# 파이썬 기초문법

## 파이썬 반복문(Loop) 기초 문법 정리

반복문은 **같은 작업을 여러 번 자동으로 수행**할 때 사용하는 문법입니다.

파이썬에서 가장 많이 사용하는 반복문은 다음과 같습니다.

* `for`
* `while`
* `range()`
* `break`
* `continue`

---

## 1. for 문

`for` 문은 **정해진 횟수** 또는 **데이터 목록을 하나씩 꺼내서 반복**할 때 사용합니다.

## 기본 구조

```python
for 변수 in 반복대상:
    실행코드
```

---

## for 문 예제

### 예제 1 : 문자열 반복

```python
for ch in "python":
    print(ch)
```

### 실행 결과

```python
p
y
t
h
o
n
```

---

### 예제 2 : 리스트 반복

```python
fruits = ["apple", "banana", "orange"]

for fruit in fruits:
    print(fruit)
```

---

### 예제 3 : 숫자 반복(range 사용)

```python
for i in range(5):
    print(i)
```

### 결과

```python
0
1
2
3
4
```

---

### 예제 4 : 합계 구하기

```python
total = 0

for i in range(1, 11):
    total += i

print(total)
```

### 결과

```python
55
```

---

### 예제 5 : 중첩 반복문

```python
for i in range(1, 4):
    for j in range(1, 4):
        print(i, j)
```

### 결과

```python
1 1
1 2
1 3
2 1
2 2
2 3
3 1
3 2
3 3
```

---

### 예제 6 : 구구단 출력

```python
for dan in range(2, 10):
    print(f"--- {dan}단 ---")

    for num in range(1, 10):
        print(f"{dan} x {num} = {dan * num}")
```

## 2. while 문

`while` 문은 **조건이 참(True)인 동안 계속 반복**합니다.

---

### 기본 구조

```python
while 조건:
    실행코드
```

---

## while 예제

### 예제 1 : 숫자 증가

```python
num = 1

while num <= 5:
    print(num)
    num += 1
```

---

### 예제 2 : 비밀번호 확인

```python
password = ""

while password != "1234":
    password = input("비밀번호 입력: ")

print("로그인 성공")
```

---

### 예제 3 : 합계 계산

```python
sum_value = 0
num = 1

while num <= 100:
    sum_value += num
    num += 1

print(sum_value)
```

---

### 예제 4 : 무한 반복 + 종료 조건

```python
while True:
    cmd = input("종료하려면 q 입력: ")

    if cmd == "q":
        break

    print("계속 실행 중")
```

---

### 예제 5 : 메뉴 시스템

```python
while True:
    print("1. 조회")
    print("2. 등록")
    print("3. 종료")

    menu = input("메뉴 선택: ")

    if menu == "1":
        print("조회 기능")

    elif menu == "2":
        print("등록 기능")

    elif menu == "3":
        print("프로그램 종료")
        break

    else:
        print("잘못 입력")
```

## 3. range() 함수

`range()` 는 숫자 범위를 생성합니다.

주로 `for` 문과 함께 사용됩니다.

---

## range 기본 형태

## 1개 값(0 ~ 끝 전까지 반복)

```python
range(5)
```

→ 0 ~ 4

---

## 2개 값(시작 ~ 끝 전까지 반복)

```python
range(1, 6)
```

→ 1 ~ 5

---

## 3개 값(시작 ~ 끝 전까지 반복, 스텝)

```python
range(1, 10, 2)
```

→ 1, 3, 5, 7, 9

---

## range 예제

### 짝수 출력

```python
for i in range(2, 11, 2):
    print(i)
```

---

### 역순 출력

```python
for i in range(10, 0, -1):
    print(i)
```

---

## 4. break 문

`break` 는 반복문을 **즉시 종료**합니다.

---

### break 예제

```python
for i in range(10):
    if i == 5:
        break

    print(i)
```

### 결과

```python
0
1
2
3
4
```

---

## 특정 값 찾기

```python
numbers = [3, 7, 2, 9, 5]

for num in numbers:
    if num == 9:
        print("찾았다!")
        break
```

---

## 로그인 제한 시스템

```python
count = 0

while count < 3:
    password = input("비밀번호 입력: ")

    if password == "1234":
        print("로그인 성공")
        break

    count += 1
    print("실패")

if count == 3:
    print("계정 잠금")
```

## 5. continue 문

`continue` 는 아래 코드를 건너뛰고 다음 반복으로 이동합니다.

---

## continue 예제

```python
for i in range(5):
    if i == 2:
        continue

    print(i)
```

### 결과

```python
0
1
3
4
```

---

## 홀수만 출력

```python
for i in range(1, 11):
    if i % 2 == 0:
        continue

    print(i)
```

---

## 특정 데이터 제외 처리

```python
scores = [90, -1, 85, 100, -1, 70]

for score in scores:

    if score == -1:
        continue

    print(f"점수: {score}")
```
