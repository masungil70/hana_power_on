# 파이썬 기초문법

## 파이썬 문자열 다루기 (String) 기초 문법 정리

파이썬에서 문자열(String)은 문자들의 집합입니다.
웹 개발, AI, 데이터 처리, 자동화 등 거의 모든 프로그램에서 가장 많이 사용하는 데이터 타입입니다.

```python
name = "Python"
msg = '안녕하세요'
```

---

## 1. 문자열(String)이란?

문자들의 묶음입니다.

```python
text = "Hello"
```

각 문자에는 번호(Index)가 존재합니다.

| 문자  | H | e | l | l | o |
| --- | - | - | - | - | - |
| 인덱스 | 0 | 1 | 2 | 3 | 4 |

---

## 2. 문자열 슬라이싱 (Slicing)

슬라이싱은 문자열의 일부를 잘라오는 기능(배열표기법)입니다.


형식:

```python
문자열[start:end]
```

* start **포함**
* end **미포함**
* 수학 수식 : start <= x < end

---

>
---

### 2-1. 기본 슬라이싱

```python
text = "Python"

print(text[0:2])
```

결과:

```python
Py
```

---

### 2-2. 처음부터 가져오기

```python
text = "Python"

print(text[:3])
```

결과:

```python
Pyt
```

---

### 2-3. 끝까지 가져오기

```python
text = "Python"

print(text[2:])
```

결과:

```python
thon
```

---

### 2-4. 마지막 문자 가져오기

```python
text = "Python"

print(text[-1])
```

결과:

```python
n
```

---

### 2-5. 간격(step) 사용하기

형식:

```python
문자열[start:end:step]
```

```python
text = "ABCDEFGHIJ"

print(text[::2])
```

결과:

```python
ACEGI
```

---

### 2-6. 문자열 뒤집기

```python
text = "Python"

print(text[::-1])
```

결과:

```python
nohtyP
```

---

### 2-7. 주민번호 일부 가리기

```python
jumin = "990101-1234567"

print(jumin[:8] + "*******")
```

결과:

```python

```

## 3. 문자열 함수

문자열 함수는 문자열을 쉽게 처리하는 기능입니다.

---

### 3-1. upper() → 대문자

```python
text = "python"

print(text.upper())
```

결과:

```python
PYTHON
```

---

### 3-2. lower() → 소문자

```python
text = "PYTHON"

print(text.lower())
```

결과:

```python
python
```

---

### 3-3. count() → 문자 개수

```python
text = "banana"

print(text.count("a"))
```

결과:

```python
3
```

---

### 3-4. len() → 문자열 길이

```python
text = "Python"

print(len(text))
```

결과:

```python
6
```

---

### 3-5. find()

문자의 위치 찾기

```python
text = "Python"

print(text.find("t"))
```

결과:

```python
2
```

못 찾으면:

```python
-1
```

---

### 3-6. index()

```python
text = "Python"

print(text.index("t"))
```

결과:

```python
2
```

차이점:

* find() → 없으면 -1
* index() → 없으면 오류 발생

---

### 3-7. startswith() : 특정문자로 시작하는지 확인

```python
text = "python.py"

print(text.startswith("python"))
```

결과:

```python
True
```

---

### 3-8. endswith() : 특정문자로 끝나는지 확인

```python
text = "python.py"

print(text.endswith(".py"))
```

결과:

```python
True
```

## 4. split()

문자열을 분리하여 리스트로 만드는 함수

---

```python
text = "apple banana orange"

result = text.split()

print(result)
```

결과:

```python
['apple', 'banana', 'orange']
```

---

### 4-1. 특정 문자 기준 분리

```python
data = "a,b,c,d"

result = data.split(",")

print(result)
```

결과:

```python
['a', 'b', 'c', 'd']
```

---

### 4-2. CSV 데이터 처리

```python
csv = "김철수,20,서울"

name, age, city = csv.split(",")

print(name)
print(age)
print(city)
```

## 5. join()

리스트를 문자열로 합치기

---

```python
words = ["Python", "Java", "C"]

result = " ".join(words)

print(result)
```

결과:

```python
Python Java C
```

---

### 5-1. 특정 문자(쉼표)로 연결

```python
nums = ["1", "2", "3"]

print(",".join(nums))
```

결과:

```python
1,2,3
```

---

### 5-2. 파일 경로 생성

```python
paths = ["home", "user", "documents"]

result = "/".join(paths)

print(result)
```

결과:

```python
home/user/documents
```

## 6. replace()

---

```python
text = "I like Java"

print(result)
```

결과:

```python
I like Python
```

### 6-2. 특정 문자("-") 제거

```python

```

결과:

```python
01012345678
```

### 6-3. 비밀번호 마스킹

## 7. strip()

### 7-1. 공백 제거 함수

---

```python
text = "   hello   "

print(text.strip())
```

결과:

```python
hello
```

---

### 7-2. 왼쪽/오른쪽 공백 제거

```python
text = "   hello   "

# 왼쪽 공백 제거
print(text.lstrip())

# 오른쪽 공백 제거
print(text.rstrip())
```

---

### 7-3. 입력 데이터 정리

```python
data = "  python@gmail.com \n"

clean = data.strip()

print(clean)
```

결과:

```python
python@gmail.com
```
