# 파이썬 기초문법

## 파이썬 함수(Function) 기초 문법 정리

함수(Function)는 **반복되는 코드를 하나로 묶어 재사용**할 수 있게 해주는 기능입니다.
프로그램이 커질수록 함수 사용은 필수입니다.

예를 들어:

* 계산 기능
* 로그인 처리
* 파일 저장
* 데이터 분석
* AI 모델 호출

등을 모두 함수로 만들 수 있습니다.

---

## 1. 함수(Function)란?

함수는 특정 작업을 수행하는 코드 묶음입니다.

예시:

```python
def hello():
    print("안녕하세요")
```

실행:

```python
hello()
```

결과:

```python
안녕하세요
```

---

### 1-1. 함수의 기본 구조

```python
def 함수이름():
    실행코드
```

예시:

```python
def test():
    print("함수 실행")
```

---

### 1-2. 함수 사용 이유

| 이유          | 설명                           |
| -------       | -----------------------------  |
| 코드 재사용   | 같은 코드를 반복 작성하지 않음 |
| 유지보수 편리 | 수정할 때 함수만 수정          |
| 가독성 향상   | 코드 구조가 깔끔해짐           |
| 모듈화        | 기능별 분리 가능               |

---

## 2. 함수 정의 (def)

기본 함수 만들기

```python
def hello():
    print("Hello Python")
```

호출:

```python
hello()
```

결과:

```python
Hello Python
```

---

## 3. 매개변수(Parameter)

매개변수는 함수에 값을 전달하는 변수입니다.

### 3-1. 매개변수 예제

```python
def hello(name):
    print(name + "님 안녕하세요")
```

실행:

```python
hello("홍길동")
```

결과:

```python
홍길동님 안녕하세요
```

---

### 3-2. 여러 개의 매개변수

```python
def add(a, b):
    print(a + b)
```

실행:

```python
add(10, 20)
```

결과:

```python
30
```

---

## 4. return

`return`은 함수의 결과값을 반환합니다.

### 4-1. return 없는 함수

```python
def add(a, b):
    print(a + b)
```

```python
result = add(3, 4)
print(result)
```

결과:

```python
7
None
```

---

### 4-2. return 있는 함수

```python
def add(a, b):
    return a + b
```

```python
result = add(3, 4)
print(result)
```

결과:

```python
7
```

---

### 4-3. return 특징

#### 4-3-1. 함수 종료 기능

```python
def test():
    print("1")
    return
    print("2")
```

결과:

```python
1
```

---

#### 4-3-2. 여러 값 반환 가능

```python
def calc(a, b):
    return a + b, a - b
```

```python
x, y = calc(10, 5)

print(x)
print(y)
```

결과:

```python
15
5
```

## 5. 기본값(Default Parameter)

매개변수에 기본값을 설정할 수 있습니다.

### 5-1. 기본값 예제

```python
def hello(name="손님"):
    print(name + "님 환영합니다")
```

실행:

```python
hello()
hello("철수")
```

결과:

```python
손님님 환영합니다
철수님 환영합니다
```

---

### 5-2. 여러 기본값

```python
def power(on=True):
    if on:
        print("전원 ON")
    else:
        print("전원 OFF")
```

```python
power()
power(False)
```

결과:

```python
전원 ON
전원 OFF
```

## 6. 지역변수(Local Variable)

함수 안에서 생성된 변수입니다.

함수 내부에서만 사용 가능합니다.

## 예제

```python
def test():
    x = 10
    print(x)
```

```python
test()
print(x)
```

결과:

```python
10
NameError 발생
```

왜냐하면:

```python
x
```

는 함수 내부에서만 존재하기 때문입니다.

---

## 7. 전역변수(Global Variable)

함수 밖에서 생성된 변수입니다.

프로그램 전체에서 사용 가능합니다.

### 7-1. 예제

```python
x = 100

def test():
    print(x)
```

```python
test()
```

결과:

```python
100
```

---

## 8. global 키워드

함수 내부에서 전역변수를 수정하려면 `global` 사용

### 8-1. 예제

```python
count = 0

def increase():
    global count
    count += 1
```

```python
increase()
increase()

print(count)
```

결과:

```python
2
```

### 9-3. 계산기 함수(여러 값 반환)

```python
def calc(a, b):
    add_result = a + b
    sub_result = a - b
    mul_result = a * b
    div_result = a / b

    return add_result, sub_result, mul_result, div_result
```

```python
a, s, m, d = calc(10, 5)

print("덧셈:", a)
print("뺄셈:", s)
print("곱셈:", m)
print("나눗셈:", d)
```

### 9-4. 홀수 짝수 판별

```python
def is_even(number):
    if number % 2 == 0:
        return True
    else:
        return False
```

```python
print(is_even(10))
print(is_even(7))
```

결과:

```python
True
False
```

---

### 9-4 리스트 평균 계산

```python
def average(numbers):
    total = sum(numbers)
    count = len(numbers)

    return total / count
```

```python
nums = [10, 20, 30, 40]

print(average(nums))
```

---

### 9-5. 가변 매개변수 (*args)

개수가 정해지지 않은 값을 받을 수 있음

```python
def total(*numbers):
    result = 0

    for n in numbers:
        result += n

    return result
```

```python
print(total(1, 2, 3))
print(total(10, 20, 30, 40))
```

결과:

```python
6
100
```

---

### 9-6.  키워드 매개변수 (**kwargs)

```python
def user_info(**data):
    for key, value in data.items():
        print(key, ":", value)
```

```python
user_info(name="홍길동", age=20, city="서울")
```

결과:

```python
name : 홍길동
age : 20
city : 서울
```

---

### 9-7. 재귀 함수(팩토리얼 예제)

함수가 자기 자신을 호출하는 함수

```python
def factorial(n):
    if n == 1:
        return 1

    return n * factorial(n - 1)
```

```python
print(factorial(5))
```

계산 과정:

```python
5 * 4 * 3 * 2 * 1
```

결과:

```python
120
```

## 10. 함수 설계 팁

| 좋은 방법             | 설명                    |
| ------------          | ----------------------- |
| 함수는 하나의 기능만  | 역할 분리               |
| 이름 명확하게         | add_user(), save_file() |
| return 적극 활용      | 결과 재사용 가능        |
| 너무 긴 함수 금지     | 유지보수 어려움         |
| 중복 제거             | 같은 코드 반복 금지     |
