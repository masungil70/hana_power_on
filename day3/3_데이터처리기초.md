# 데이터 처리

**“데이터를 어떻게 다루는가”** 에 대한 사고방식을 익히는 방법을 학습 하겠습니다.

즉, 단순한 용어을 학습하는 수준이 아니라:

* 데이터를 읽고
* 정리하고
* 변환하고
* 분석 가능한 형태로 만들고
* 시각화 및 AI/ML에 연결

하는 흐름을 이해하고 처리하는 방법을 학습할 것입니다.

---

## 1. 데이터 처리(Data Processing)란?

데이터 처리는:

> “원본 데이터(raw data)를 의미 있는 정보로 변환하는 과정”

입니다.

예시:

| 원본 데이터      | 처리 후          |
| ---------        | ---------        |
| 쇼핑몰 주문 로그 | 월별 매출 통계   |
| 센서 데이터      | 이상 탐지        |
| 이미지 픽셀      | 객체 인식        |
| 웹 로그          | 사용자 행동 분석 |

---

## 2. 데이터 처리 전체 흐름

```text
데이터 수집
   ↓
데이터 저장
   ↓
데이터 정제(Cleaning)
   ↓
데이터 변환(Transformation)
   ↓
데이터 분석(Analysis)
   ↓
데이터 시각화(Visualization)
   ↓
AI / 머신러닝 활용
```

앞으로 학습하는 NumPy, Pandas, Matplotlib도 결국 이 흐름 안에서 사용됩니다.

---

## 3. 데이터 처리 핵심 개념

### 1단계. 데이터(Data)의 종류 이해

#### 데이터(Data)의 종류

1. 정형 데이터(Structured Data)
2. 비정형 데이터(Unstructured Data)
3. 반정형 데이터(Semi-Structured)

---

### 1 정형 데이터(Structured Data)

표 형태 데이터로 일정한 구조가 있습니다.

예:

| 이름 | 나이 | 점수 |
| ---  | --   | --   |
| Kim  | 20   | 90   |
| Hong | 50   | 90   |

대표 저장 형식:

* CSV
* Excel
* SQL DB

→ Pandas의 핵심 대상이 됩니다

---

### 2. 비정형 데이터(Unstructured Data)

형태가 일정하지 않은 데이터를 의미합니다

예:

* 이미지
* 영상
* 음성
* 텍스트

→ AI/딥러닝에서 중요합니다

---

### 3. 반정형 데이터(Semi-Structured)

완전한 표는 아니지만 구조 있음

예:

* JSON
* XML
* YAML

예시:

```json
{
  "name": "kim",
  "age": 20
}
```

→ API 처리에서 매우 중요하게 사용됩니다

---

### 2단계. 데이터 처리의 핵심 목적 이해

데이터 처리는 결국 아래 4개가 핵심입니다.

| 목적 | 설명                     |
| --   | ----------------         |
| 수집 | 파일/API/DB에서 가져오기 |
| 정리 | 오류 제거                |
| 변환 | 원하는 형태로 변경       |
| 분석 | 의미 도출                |

### 3단계. 데이터 정제(Cleaning) 개념

데이터 정제는 실무에서 가장 중요합니다.

실제 데이터는 거의 항상 깨끗하지 않습니다.

예:

```text
홍길동,20
홍길동 , 20
hong,NaN
NULL
```

문제점:

* 공백
* 결측치
* 중복
* 형식 불일치

---

### 반드시 알아야 하는 데이터 정제 개념

| 개념                  | 설명             |
| ------------------    | ---------------- |
| 결측치(Missing Value) | 값이 없음        |
| 중복 데이터           | 같은 데이터 반복 |
| 이상치(Outlier)       | 비정상 값        |
| 타입 변환             | 문자열 → 숫자    |
| 공백 제거             | trim             |
| 정규화                | 데이터 형식 통일 |

### CSV 기본 처리

```python
import csv

with open("data.csv") as f:
    # CSV 파일 전체를 한번에 읽어서 list로 구성합니다 
    reader = csv.reader(f)

    # list에서 한 줄씩 읽어서 출력 합니다 
    for row in reader:
        print(row)
```

### JSON 처리

#### 1. JSON 파일을 읽어서 내용 출력하기

```python
# read_json.py

import json

# sample.json 파일 읽기
with open("sample.json", "r", encoding="utf-8") as file:
    data = json.load(file)

# 전체 내용 출력
print("JSON 전체 데이터:")
print(data)

# 특정 값 출력
print("\n이름:", data["name"])
print("나이:", data["age"])
print("기술 스택:", data["skills"])
```

#### sample.json

```json
{
    "name": "Kim",
    "age": 30,
    "skills": ["Python", "FastAPI", "Docker"]
}
```

#### 실행 결과

```bash
JSON 전체 데이터:
{'name': 'Kim', 'age': 30, 'skills': ['Python', 'FastAPI', 'Docker']}

이름: Kim
나이: 30
기술 스택: ['Python', 'FastAPI', 'Docker']
```

---

#### 2. JSON 문자열을 Dictionary로 변환하기

```python
# json_string_to_dict.py

import json

# JSON 문자열
json_string = '''
{
    "server": "nginx",
    "port": 80,
    "enable": true
}
'''

# 문자열 -> Dictionary 변환
data = json.loads(json_string)

# 타입 확인
print(type(data))

# 데이터 출력
print("전체 데이터:", data)
print("서버:", data["server"])
print("포트:", data["port"])
print("활성화:", data["enable"])
```

#### 실행 결과

```bash
<class 'dict'>
전체 데이터: {'server': 'nginx', 'port': 80, 'enable': True}
서버: nginx
포트: 80
활성화: True
```

---

#### 3. Dictionary를 JSON 파일로 저장하기

```python
# dict_to_json_file.py

import json

# Dictionary 데이터
data = {
    "name": "Lee",
    "age": 28,
    "city": "Seoul",
    "skills": ["Python", "Kubernetes", "AWS"]
}

# Dictionary -> JSON 파일 저장
with open("output.json", "w", encoding="utf-8") as file:
    json.dump(data, file, ensure_ascii=False, indent=4)

print("output.json 파일 저장 완료")
```

#### 생성되는 output.json

```json
{
    "name": "Lee",
    "age": 28,
    "city": "Seoul",
    "skills": [
        "Python",
        "Kubernetes",
        "AWS"
    ]
}
```

### 7단계. 데이터 변환(Transformation)

데이터를 원하는 형태로 바꾸는 과정.

---

#### 예시

```python
# 원으로된 가격 데이터
prices = [1000, 2000, 3000]

# 원 → 달러 변환 비율
exchange_rate = 1500
dollars = []

for p in prices:
    dollars.append(p / exchange_rate)
```

---

#### 변환 핵심 개념

| 개념        | 설명      |
| ----------- | -----     |
| Filtering   | 조건 추출 |
| Mapping     | 값 변환   |
| Aggregation | 합계/평균 |
| Sorting     | 정렬      |
| Grouping    | 그룹화    |

이것이 나중에 Pandas 핵심 기능이 됩니다.

### 9단계. 함수(Function) 기반 처리

실무에서는 함수화가 중요합니다.

---

## 예시

```python
# 앞뒤 공백 제거 및 소문자로 변환
# 함수 선언 및 구현 
def clean_name(name):
    return name.strip().lower()

# 앞뒤 공백 있고 대문자와 소문자로 구성된 자료 전달하여 호출 
print(clean_name(" Kim "))



# Using map and lambda
# 정수와 문자열로 구성된 자료를 전달하여 호출 정수로 변환하여 * 2를 한다
nums = [1," 2 ",3," 4 "]
print(list(map(lambda x: int(str(x).strip()) * 2, nums)))


```
