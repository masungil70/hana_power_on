# 파이썬 기초문법

## 파이썬 리스트(List) 기초 문법 정리

리스트(List)는 여러 개의 데이터를 하나의 변수에 저장할 수 있는 자료형입니다.
파이썬에서 가장 많이 사용하는 자료구조 중 하나이며, 데이터 모음 관리의 기본입니다.

---

## 1. 리스트(List)란?

리스트는 여러 값을 순서대로 저장하는 자료형입니다.

```python
fruits = ["apple", "banana", "orange"]
```

특징:

* 여러 데이터를 저장 가능
* 순서(index)가 있음
* 수정 가능(mutable)
* 중복 허용

---

## 2. 리스트 생성

### 기본 생성

```python
numbers = [1, 2, 3, 4, 5]

print(numbers)
```

출력:

```python
[1, 2, 3, 4, 5]
```

---

### 문자열 리스트

```python
names = ["kim", "lee", "park"]

print(names)
```

---

### 다양한 자료형 저장 가능

```python
data = [10, "hello", True, 3.14]

print(data)
```

---

## 3. append() 함수

리스트 마지막에 데이터를 추가합니다.

## 기본 사용

```python
fruits = ["apple", "banana"]

fruits.append("orange")

print(fruits)
```

출력:

```python
['apple', 'banana', 'orange']
```

---

### 숫자 추가

```python
numbers = [1, 2, 3]

numbers.append(4)

print(numbers)
```

---

## 4. remove() 함수

리스트에서 특정 값을 삭제합니다.

### 4-1. 기본 사용법(값으로 삭제)

```python
fruits = ["apple", "banana", "orange"]

fruits.remove("banana")

print(fruits)
```

출력:

```python
['apple', 'orange']
```

---

### 4-2 삭제 오류 발생 예시

존재하지 않는 값을 삭제하면 오류 발생

```python
numbers = [1, 2, 3]

numbers.remove(5)
```

출력:

```python
ValueError: list.remove(x): x not in list 
```

`remove()`는 리스트 안에 존재하는 값을 삭제할 때 사용합니다.
하지만 `5`는 리스트에 없기 때문에 오류가 발생합니다.

### 4-3 삭제시 if in 사용 방법

삭제하려는 값이 있는지 먼저 확인

```python
numbers = [1, 2, 3]

if 5 in numbers:
    numbers.remove(5)

print(numbers)
```

출력:

```python
[1, 2, 3]
```

---

### 4-4 삭제시 예외 처리 사용 방법

예외 처리 사용 (`try-except`)

```python
numbers = [1, 2, 3]

try:
    numbers.remove(5)
except ValueError:
    print("삭제할 값이 리스트에 없습니다.")

print(numbers)
```

출력:

```python
삭제할 값이 리스트에 없습니다.
[1, 2, 3]
```

---

## 5. 인덱스(Index)

리스트의 위치 번호입니다.

파이썬은 0부터 시작합니다.

| 값     | 인덱스 |
| ------ | ---    |
| apple  | 0      |
| banana | 1      |
| orange | 2      |

---

### 5-1. 색인(인덱스)으로 데이터 접근

```python
fruits = ["apple", "banana", "orange"]

print(fruits[0])
print(fruits[1])
```

출력:

```python
apple
banana
```

---

### 5-2. 음수 인덱스

뒤(오른쪽)에서부터 접근 가능

```python
fruits = ["apple", "banana", "orange"]

print(fruits[-1])
```

출력:

```python
orange
```

---

## 6. 슬라이싱(Slicing)

리스트의 일부를 잘라오는 기능입니다.

형식:

```python
리스트[start:end]
```

* start 포함
* end 미포함

---

### 6-1. 기본 슬라이싱

```python
numbers = [1, 2, 3, 4, 5]

print(numbers[1:4])
```

출력:

```python
[2, 3, 4]
```

---

### 6-2. 처음부터 가져오기

```python
print(numbers[:3])
```

출력:

```python
[1, 2, 3]
```

---

### 6-3. 끝까지 가져오기

```python
print(numbers[2:])
```

출력:

```python
[3, 4, 5]
```

---

### 6-4. 간격(step) 지정

```python
numbers = [1,2,3,4,5,6,7,8]

print(numbers[::2])
```

출력:

```python
[1, 3, 5, 7]
```

---

## 7. 반복문과 리스트

리스트는 반복문과 함께 매우 많이 사용됩니다.

---

### 7-1. for문 사용

```python
fruits = ["apple", "banana", "orange"]

for fruit in fruits:
    print(fruit)
```

출력:

```python
apple
banana
orange
```

---

### 7-2. range() 함수와 인덱스 함께 사용

```python
fruits = ["apple", "banana", "orange"]

for i in range(len(fruits)):
    print(i, fruits[i])
```

출력:

```python
0 apple
1 banana
2 orange
```
