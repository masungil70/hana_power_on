# 파이썬 기초문법

## 파이썬 컴프리헨션(List Comprehension) 기초 문법 정리

파이썬의 **컴프리헨션(Comprehension)** 은
기존 리스트나 반복 가능한 객체(iterable)를 기반으로 **짧고 간결하게 새로운 리스트를 생성하는 문법**입니다.

기존의 `for` 문보다 코드가 짧고 가독성이 좋아서 실무에서도 많이 사용됩니다.

일반 Python 리스트, 딕셔너리(Dictionary), 집합(Set)를 대상으로 사용합니다.

---

## 1. 기본 문법

```python
[표현식 for 변수 in 반복가능한객체]
```

예제:

```python
numbers = [1, 2, 3, 4, 5]

result = [x * 2 for x in numbers]

print(result)
```

결과:

```python
[2, 4, 6, 8, 10]
```

---

## 2. 일반 for문과 비교

### 일반 for문

```python
numbers = [1, 2, 3, 4, 5]

result = []

for x in numbers:
    result.append(x * 2)

print(result)
```

---

### 리스트 컴프리헨션

```python
numbers = [1, 2, 3, 4, 5]

result = [x * 2 for x in numbers]

print(result)
```

더 짧고 직관적입니다.

---

## 3. 조건문 사용하기

### 짝수만 추출

```python
numbers = [1, 2, 3, 4, 5, 6]

result = [x for x in numbers if x % 2 == 0]

print(result)
```

결과:

```python
[2, 4, 6]
```

---

## 4. if-else 사용하기

문법:

```python
[참일때값 if 조건 else 거짓일때값 for 변수 in 반복객체]
```

예제:

```python
numbers = [1, 2, 3, 4, 5]

result = ["짝수" if x % 2 == 0 else "홀수" for x in numbers]

print(result)
```

결과:

```python
['홀수', '짝수', '홀수', '짝수', '홀수']
```

---

## 5. 문자열 처리 예제

### 문자열을 대문자로 변환

```python
words = ["apple", "banana", "orange"]

# 소문자 리스트를 대문자로 변환
result = [word.upper() for word in words]

print(result)
```

결과:

```python
['APPLE', 'BANANA', 'ORANGE']
```

---

## 6. 중첩 리스트 컴프리헨션

### 2차원 리스트 생성

```python
# 3x3 행렬 생성
matrix = [[x for x in range(3)] for y in range(3)]

print(matrix)
```

결과:

```python
[
 [0, 1, 2],
 [0, 1, 2],
 [0, 1, 2]
]
```

---

## 7. 실무 스타일 예제

### 파일 확장자 필터링

```python
files = ["a.txt", "b.jpg", "c.txt", "d.png"]

# txt 파일만 추출
txt_files = [file for file in files if file.endswith(".txt")]

print(txt_files)
```

결과:

```python
['a.txt', 'c.txt']
```

---

## 8. 리스트 컴프리헨션의 장점

* 코드가 짧아짐
* 가독성 향상
* 빠른 리스트 생성
* 함수형 스타일 코드 가능

---

## 9. 주의사항

너무 복잡하게 중첩하면 오히려 가독성이 떨어집니다.

지나치게 복잡한 예:

```python
result = [[x*y for x in range(5) if x % 2 == 0] for y in range(5)]
```

이런 경우는 일반 `for` 문이 더 읽기 쉽습니다.

## 10. 자주 사용하는 패턴 정리

### 10-1. 값 변환

```python
[x * 2 for x in data]
```

---

### 10-2. 조건 필터링

```python
[x for x in data if x > 10]
```

---

### 10-3. 조건 분기

```python
["YES" if x > 0 else "NO" for x in data]
```

---

### 10-4. 문자열 처리

```python
[word.upper() for word in words]
```

## 11. 학생 점수 처리 예제

```python
scores = [85, 42, 77, 90, 35]

# 점수가 60점 이상이면 "합격", 아니면 "불합격" 처리
passed = ["합격" if score >= 60 else "불합격" for score in scores]

print(passed)
```

결과:

```python
['합격', '불합격', '합격', '합격', '불합격']
```

---

## 12. 딕셔너리 컴프리헨션

리스트뿐 아니라 딕셔너리도 가능합니다.

```python
numbers = [1, 2, 3, 4]

result = {x: x*x for x in numbers}

print(result)
```

결과:

```python
{
 1: 1,
 2: 4,
 3: 9,
 4: 16
}
```

---

## 13. 집합(Set) 컴프리헨션

```python
numbers = [1, 1, 2, 2, 3, 3]

result = {x for x in numbers}

print(result)
```

결과:

```python
{1, 2, 3}
```

---

## 14. 정리

리스트 컴프리헨션은 다음 상황에서 매우 유용합니다.

* 데이터 변환
* 조건 필터링
* 문자열 가공
* 파일 처리
* AI/데이터 분석 전처리
* FastAPI/백엔드 데이터 가공

특히 AI/데이터 처리에서는 거의 필수적으로 사용됩니다.

예:

```python
normalized = [x / 255.0 for x in image_pixels]
```

또는:

```python
valid_users = [u for u in users if u["active"]]
```
