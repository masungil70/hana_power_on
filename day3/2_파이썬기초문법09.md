# 파이썬 기초문법

## 파이썬 파일 처리(File Handling) 기초 문법 정리

파일 처리는 **실무에서 매우 중요한 기능**입니다.
왜냐하면 대부분의 프로그램은 데이터를 저장하거나 읽어와야 하기 때문입니다.

예:

* 로그 저장
* 회원 정보 저장
* 설정 파일 읽기
* CSV/JSON 데이터 처리
* AI 학습 데이터 저장
* 웹 서버 로그 분석

파이썬에서는 `open()` 함수를 이용하여 파일을 읽고 쓸 수 있습니다.

---

## 1. 파일 처리란?

파일(File)은 디스크에 저장되는 데이터입니다.

예:

* `.txt`
* `.csv`
* `.json`
* `.log`

파이썬은 파일을:

* 읽기(Read)
* 쓰기(Write)
* 추가(Append)
* 수정(Update)

할 수 있습니다.

---

## 2. 기본 구조

```python
파일객체 = open("파일명", "모드")
```

예:

```python
f = open("test.txt", "r")
```

---

## 3. 파일 모드 종류

| 모드 | 의미          |
| --   | ----------    |
| r    | 읽기(Read)    |
| w    | 쓰기(Write)   |
| a    | 추가(Append)  |
| x    | 새 파일 생성  |
| t    | 텍스트 모드   |
| b    | 바이너리 모드 |

---

## 4. 가장 중요한 with open()

실무에서는 거의 항상 이것을 사용합니다.

```python
with open("파일명", "모드") as 변수:
    코드
```

이유:

* 자동으로 파일 닫힘
* 메모리 안전
* 오류 감소
* 실무 표준 방식

---

## 5. 파일 읽기

---

### 5-1. 파일 전체 읽기 : read()

파일명 : test.txt

내용 :
```txt
Hello
Python
```

```python
with open("test.txt", "r") as f:
    data = f.read()

print(data)
```

---

### 5-2. 파일 한 줄 읽기 : readline()

```python
with open("test.txt", "r") as f:
    line = f.readline()

print(line)
```

---

### 5-3. 파일의 모든 줄을 리스트로 읽기 : readlines()

```python
with open("test.txt", "r") as f:
    lines = f.readlines()

print(lines)
```

출력:

```python
['Hello\n', 'Python\n']
```

---

## 6. 파일 쓰기

---

### 6-1. write()

```python
with open("test.txt", "w") as f:
    f.write("안녕하세요")
```

주의:

* 기존 내용 삭제 후 새로 작성됨

---

## 7. 파일 추가(Append)

```python
with open("test.txt", "a") as f:
    f.write("\n새로운 내용")
```

기존 내용 유지됨

## 8. UTF-8 인코딩 (실무 필수)

한글 깨짐 방지

```python
with open("test.txt", "r", encoding="utf-8") as f:
    data = f.read()
```

실무에서는 거의 항상 사용합니다.

---

## 9. 예제

### 9-1. 파일 생성 후 저장

```python
with open("hello.txt", "w", encoding="utf-8") as f:
    f.write("사과\n")
    f.write("바나나\n")
    f.write("포도\n")
```

결과:

* hello.txt 생성됨

---

### 9-2. 파일 읽기

```python
with open("hello.txt", "r", encoding="utf-8") as f:
    text = f.read()

print(text)
```

출력:

```python
사과
바나나
포도
```

---

### 9-3. 여러 줄 저장

```python
with open("data.txt", "w", encoding="utf-8") as f:
    f.write("사과\n")
    f.write("바나나\n")
    f.write("포도\n")
```

---

### 9-4. 줄 단위 출력

파일명 : data.txt

내용 :
```txt
사과
바나나
포도
```

```python
with open("data.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())
```

출력:

```python
사과
바나나
포도
```

---

### 9-5. 사용자 입력 저장

```python
name = input("이름 입력: ")

with open("user.txt", "w", encoding="utf-8") as f:
    f.write(name)
```

---

### 9-6. 로그 파일 만들기

```python
with open("app.log", "a", encoding="utf-8") as f:
    f.write("프로그램 실행됨\n")
```

### 9-7. 파일 복사

```python
#파일 내용 한번에 읽기 
with open("source.txt", "r", encoding="utf-8") as src:
    content = src.read()

#파일에 내용 기록 
with open("copy.txt", "w", encoding="utf-8") as dst:
    dst.write(content)
```

---

### 9-8. 단어 개수 세기

```python
with open("text.txt", "r", encoding="utf-8") as f:
    data = f.read()

words = data.split()

print("단어 수:", len(words))
```

---

### 9-9. 특정 문자열 찾기

```python
with open("text.txt", "r", encoding="utf-8") as f:
    data = f.read()

if "Python" in data:
    print("Python 발견")
else:
    print("없음")
```

---

### 9-10. CSV 저장

```python
data = [
    "이름,나이",
    "홍길동,20",
    "김철수,25"
]

with open("users.csv", "w", encoding="utf-8") as f:
    for line in data:
        f.write(line + "\n")
```

---

### 9-11. CSV 읽기

```python
with open("users.csv", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip().split(","))
```

출력:

```python
['이름', '나이']
['홍길동', '20']
['김철수', '25']
```

---

### 9-12. 에러 처리 포함

```python
try:
    with open("none.txt", "r", encoding="utf-8") as f:
        data = f.read()

    print(data)

except FileNotFoundError:
    print("파일이 존재하지 않습니다.")
```

### 9-13. 파일 자동 백업

```python
import shutil

#data.txt파일을 backup.txt로 복사 합니다 
shutil.copy("data.txt", "backup.txt")

print("백업 완료")
```

---

### 9-14. 대용량 파일 처리

```python

#파일에 1줄씩 읽어서 가공하고 기록합니다 
#파일을 한번에 읽어서 처리 하려만 많은 메모리가 필요합니다 
with open("big.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())
```

좋은 이유:

* 메모리 절약
* 속도 향상
