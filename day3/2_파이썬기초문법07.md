# 파이썬 기초문법

## 파이썬 람다(lambda) 기초 문법 정리

람다 함수(lambda function)는 **이름 없이 한 줄로 만드는 간단한 함수**입니다.

주로 다음 상황에서 많이 사용합니다.

* `sort()`, `map()`, `filter()` 와 함께 사용할 때
* 데이터 처리 시 임시 함수가 필요할 때 (데이터 처리 로직 주입)
* AI/데이터 분석 코드에서 간단한 연산 처리할 때
* callback 함수 전달
* 사용자 정의 동작(custom behavior)
* 함수형 프로그래밍 스타일
* 전략 패턴을 구현하는 도구로 사용 할 때

---

## 1. 람다 함수 기본 개념

일반 함수:

```python
def add(a, b):
    return a + b
```

람다 함수:

```python
lambda a, b: a + b
```

둘은 기능적으로 거의 동일합니다.

---

## 2. 람다 함수 구조

```python
lambda 매개변수: 반환값
```

예제:

```python
lambda x: x * 2
```

의미:

```python
def func(x):
    return x * 2
```

---

## 3. 람다 함수 특징

| 특징               | 설명                 |
| ----------         | -----------          |
| 이름 없음          | 익명 함수            |
| 한 줄만 가능       | 여러 줄 작성 불가    |
| return 생략        | 결과가 자동 반환     |
| 간단한 함수에 적합 | 복잡한 로직엔 부적합 |
| 함수 객체          | 변수에 저장 가능     |

---

## 4. 왜 사용하는가?

### 일반 함수

```python
def square(x):
    return x * x

```

```python
square = lambda x: x * x

print(square(3))
```

## 12. map() 과 lambda 사용

### 기본 개념

map(function, iterable)

모든 요소를 변환

| 매개변수 | 설명                             |
| -------- | ------------------------         |
| function | 각 요소에 적용할 함수            |
| iterable | 반복 가능한 객체(list, tuple 등) |

---

### 예제

```python
numbers = [1, 2, 3, 4, 5]

# 일반 반복문 버전
result = []

for x in numbers:
    result.append(x * 2)

print(result)

# map 버전, 위 코드와 동일한 결과
result = map(lambda x: x * 2, numbers)

print(list(result))
```

출력:

```python
[2, 4, 6, 8, 10]
```

---

## 13. filter() 와 lambda

### 기본 개념

filter(function, iterable)

조건에 맞는 데이터만 추출, for - if 문을 합친 구문

| 매개변수 | 설명                             |
| -------- | ------------------------         |
| function | 각 요소에 적용할 함수            |
| iterable | 반복 가능한 객체(list, tuple 등) |

---

### 예제

```python
numbers = [1, 2, 3, 4, 5, 6]

# 일반 반복문 버전
result = []

for x in numbers:
    if x % 2 == 0:
        result.append(x)

print(result)

# filter 버전, 위 코드와 동일한 결과
result = filter(lambda x: x % 2 == 0, numbers)

print(list(result))
```

출력:

```python
[2, 4, 6]
```

## 3. sorted() 와 lambda

람다는 정렬에서 엄청 많이 사용됩니다.

sorted(iterable, key=None, reverse=False)

| 매개변수 | 설명           |
| -------- | --------       |
| iterable | 정렬할 데이터  |
| key      | 정렬 기준 함수 |
| reverse  | 내림차순 여부  |

---

### 튜플 정렬

```python
students = [
    ("Kim", 90),
    ("Lee", 70),
    ("Park", 85)
]

result = sorted(students, key=lambda x: x[1])

print(result)
```

출력:

```python
[('Lee', 70), ('Park', 85), ('Kim', 90)]
```

---

## 4. 내림차순 정렬

```python
students = [
    ("Kim", 90),
    ("Lee", 70),
    ("Park", 85)
]

result = sorted(
    students,
    key=lambda x: x[1],
    reverse=True
)

print(result)
```

출력:

```python
[('Kim', 90), ('Park', 85), ('Lee', 70)]
```

---

## 5. 문자열 길이 정렬

```python
words = ["apple", "kiwi", "banana", "grape"]

result = sorted(words, key=lambda x: len(x))

```

출력:
