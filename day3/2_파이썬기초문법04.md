# 파이썬 기초문법

## 파이썬 튜플(Tuple) / 딕셔너리(Dictionary) / 집합(Set) 기초 문법 정리

---

## 1. 튜플 (Tuple)

튜플은 여러 개의 데이터를 저장하는 자료형입니다.
리스트와 비슷하지만 **수정이 불가능(immutable)** 하다는 특징이 있습니다.

---

### 1-1. 튜플 특징

| 특징        | 설명                        |
| ------      | ---------------------       |
| 순서 있음   | 인덱스 사용 가능            |
| 수정 불가능 | 값 변경 불가                |
| 중복 허용   | 같은 값 저장 가능           |
| 속도 빠름   | 리스트보다 메모리 효율 좋음 |

---

### 1-2. 튜플 생성

```python
numbers = (1, 2, 3)

print(numbers)
```

출력

```python
(1, 2, 3)
```

---

### 1-3. 인덱스 접근

```python
fruits = ("apple", "banana", "orange")

print(fruits[0])
print(fruits[1])
```

출력

```python
apple
banana
```

---

### 1-4. 슬라이싱

```python
numbers = (10, 20, 30, 40, 50)

print(numbers[1:4])
```

출력

```python
(20, 30, 40)
```

---

### 1-5. 튜플은 수정 불가능

```python
numbers = (1, 2, 3)

numbers[0] = 100
```

출력

```python
TypeError 발생
```

---

### 1-6. 튜플 사용 이유

* 변경되면 안 되는 데이터 저장
* 좌표값
* 설정값
* 함수 반환값

예시

```python
point = (100, 200)
```

---

### 1-7. 튜플 예제

---

#### 1-7-1. 학생 정보 저장

```python
student = ("홍길동", 20, "컴퓨터공학")

print(student)
print(student[0])
```

출력

```python
('홍길동', 20, '컴퓨터공학')
홍길동
```

---

#### 1-7-2. 튜플 반복문

```python
colors = ("red", "green", "blue")

for color in colors:
    print(color)
```

출력

```python
red
green
blue
```

---

### 1-8. 튜플 언패킹

```python
person = ("김철수", 30, "서울")

name, age, city = person

print(name)
print(age)
print(city)
```

출력

```python
김철수
30
서울
```

## 2. 딕셔너리 (Dictionary)

딕셔너리는
**key : value 형태**로 데이터를 저장합니다.

JSON 데이터와 매우 비슷합니다.

---

### 2-1. 딕셔너리 특징

| 특징             | 설명              |
| ------------     | -------------     |
| key-value 구조   | 키와 값 저장      |
| 순서 유지        | Python 3.7 이상   |
| 수정 가능        | 값 변경 가능      |
| key 중복 불가    | value는 중복 가능 |

---

### 2-2. 딕셔너리 생성

```python
student = {
    "name": "홍길동",
    "age": 20,
    "major": "컴퓨터공학"
}

print(student)
```

출력

```python
{'name': '홍길동', 'age': 20, 'major': '컴퓨터공학'}
```

---

### 2-3. 값 가져오기(읽기)

```python
print(student["name"])
print(student["age"])
```

출력

```python
홍길동
20
```

---

### 2-4. 값 수정(쓰기)

```python
student["age"] = 25

print(student)
```

출력

```python
{'name': '홍길동', 'age': 25, 'major': '컴퓨터공학'}
```

---

### 2-5. 값 추가

```python
student["grade"] = 3

print(student)
```

출력

```python
{'name': '홍길동', 'age': 25, 'major': '컴퓨터공학', 'grade': 3}
```

---

### 2-6. 값 삭제

```python
del student["major"]

print(student)
```

출력

```python
{'name': '홍길동', 'age': 25, 'grade': 3}
```

---

### 2-7. 딕셔너리 반복문

```python
student = {
    "name": "홍길동",
    "age": 20
}

for key, value in student.items():
    print(key, value)
```

출력

```python
name 홍길동
age 20
```

---

### 2-8. 딕셔너리 예제

---

#### 2-8-1. 과일 가격 저장

```python
fruits = {
    "apple": 1000,
    "banana": 2000,
    "orange": 1500
}

print(fruits["apple"])
```

출력

```python
1000
```

---

#### 2-8-2. 학생 성적 관리

```python
scores = {
    "kim": 90,
    "lee": 80,
    "park": 95
}

for name, score in scores.items():
    print(name, score)
```

출력

```python
kim 90
lee 80
park 95
```

---

#### 2-8-3. 리스트 + 딕셔너리 조합

```python
students = [
    {"name": "김철수", "age": 20},
    {"name": "이영희", "age": 21},
    {"name": "박민수", "age": 22}
]

for student in students:
    print(student["name"], student["age"])
```

출력

```python
김철수 20
이영희 21
박민수 22
```

## 3. 집합 (Set)

집합은 중복되지 않는 데이터를 저장하는 자료형입니다.

수학의 집합 개념과 매우 비슷합니다.

---

### 3-1. 집합 특징

| 특징           | 설명                     |
| --------       | --------------           |
| 중복 제거      | 같은 값 자동 제거        |
| 순서 없음      | 인덱스 사용 불가         |
| 빠른 검색      | 매우 빠른 포함 여부 확인 |
| 집합 연산 가능 | 합집합, 교집합 등        |

---

### 3-2. 집합 생성

```python
numbers = {1, 2, 3, 3, 3}

print(numbers)
```

출력

```python
{1, 2, 3}
```

---

### 3-3. 집합에 값 추가

```python
numbers.add(4)

print(numbers)
```

---

### 3-4. 집합에서 값 삭제

```python
numbers.remove(2)

print(numbers)
```

---

### 3-5. 집합 연산

```python
a = {1, 2, 3}
b = {3, 4, 5}

print(a | b)  # 합집합
print(a & b)  # 교집합
print(a - b)  # 차집합
```

출력

```python
{1, 2, 3, 4, 5}
{3}
{1, 2}
```

---

### 3-6. 집합 단계별 예제

### 3-6-1. 리스트 집합을 사용하여 중복 제거

```python
numbers = [1, 1, 2, 2, 3, 4]

result = set(numbers)

print(result)
```

출력

```python
{1, 2, 3, 4}
```

---

### 3-6-2. 공통 친구 찾기(교집합)

```python
a = {"철수", "영희", "민수"}
b = {"민수", "지영", "철수"}

common = a & b

print(common)
```

출력

```python
{'철수', '민수'}
```

### 3-6-3. 방문자 중복 체크 시스템

```python
visitors = set()

visitors.add("user1")
visitors.add("user2")
visitors.add("user1")

print(visitors)
```

출력

```python
{'user1', 'user2'}
```

## 4. 리스트 / 튜플 / 집합 비교

| 자료형 | 순서 | 수정 | 중복 |
| ---    | --   | --   | --   |
| 리스트 | O    | O    | O    |
| 튜플   | O    | X    | O    |
| 집합   | X    | O    | X    |

---

## 5. 딕셔너리 vs 집합 차이

| 자료형   | 구조        |
| ----     | ----------- |
| 딕셔너리 | key : value |
| 집합     | 값만 저장   |
