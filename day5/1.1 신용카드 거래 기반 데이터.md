# 1. **신용카드 거래 기반 데이터** 분석

파일명 : credit_card_transactions.csv

신용카드 거래 기반 데이터 파일로 하는 일은 한마디로 정리하면:

> **신용카드 거래 데이터를 분석해서 “정상 거래와 이상 거래(사기)”를 구분하고, 고객 행동 패턴을 이해하는 작업**입니다.

이 데이터는 단순한 기록이 아니라 **금융 AI/데이터 분석의 대표적인 실습용 “Fraud Detection 데이터셋”**입니다.

---

# 1. 이 파일로 하는 핵심 작업

1. 사기 거래 탐지 (Fraud Detection)
2. 고객 행동 분석 (Customer Behavior Analysis)
3. 소비 패턴 분석 (Spending Pattern)
4. 위치 기반 분석 (Geo Analysis)
5. 금융 리스크 분석 (Risk Analysis)
6. 머신러닝 / AI 모델 학습

---

## 1) 🚨 사기 거래 탐지 (Fraud Detection)

가장 중요한 목적입니다.

### 하는 일:

* 정상 거래 vs 사기 거래 분류
* `is_fraud` (0/1)을 예측하는 머신러닝 모델 학습

### 예:

* 새벽 3시 고액 결제
* 다른 도시에서 갑자기 결제
* 평소 소비 패턴과 다른 금액

👉 이런 패턴을 학습해서 “사기 가능성”을 예측합니다.

---

## 2) 👤 고객 행동 분석 (Customer Behavior Analysis)

### 하는 일:

* 고객이 언제, 어디서, 얼마나 쓰는지 분석

### 분석 예:

* 평균 소비 금액
* 소비 시간대 (낮/밤)
* 자주 사용하는 업종(category)

👉 결과:

* VIP 고객 분류
* 위험 고객 탐지

---

## 3) 📊 소비 패턴 분석 (Spending Pattern)

### 하는 일:

* 사람들의 소비 습관 분석

### 예:

* 음식 vs 쇼핑 vs 여행 비율
* 월별/시간별 소비 변화

👉 결과:

* 마케팅 전략 수립
* 맞춤형 추천 서비스

---

## 4) 📍 위치 기반 분석 (Geo Analysis)

### 하는 일:

* 고객 위치 vs 가맹점 위치 비교

### 예:

* 서울에 있는 사람이 뉴욕에서 결제
* 이동 불가능 거리 분석

👉 결과:

* 도난 카드 탐지
* 비정상 위치 거래 탐지

---

## 5) 💰 금융 리스크 분석 (Risk Analysis)

### 하는 일:

* 어떤 거래가 위험한지 점수화

### 예:

* 고액 + 새벽 + 해외 거래 = 위험 ↑

👉 결과:

* 금융 기관의 사전 차단 시스템

---

## 6) 🤖 머신러닝 / AI 모델 학습

### 하는 일:

이 데이터를 사용해서 모델을 학습합니다:

* Logistic Regression
* Random Forest
* XGBoost / LightGBM
* Isolation Forest (이상 탐지)

### 목표:

```text
입력: 거래 데이터
출력: is_fraud (0 or 1)
```
